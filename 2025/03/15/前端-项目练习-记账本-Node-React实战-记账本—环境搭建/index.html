这是因为 8.0 之前的 mysql 版本，加密规则是 mysql_native_password，而在 8.0 之后，加密规则变为 caching_sha2_password。此时你如果用的是 8.0 以前的版本，那么通过 egg 启动项目连接数据库是没问题的，我们这里使用的是 8.0 以后的版本，所以就会出现上述错误。
