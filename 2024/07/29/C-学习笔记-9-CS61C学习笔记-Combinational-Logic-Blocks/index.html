减法器设计

极巧减法：A-B = A + ( - B )

  • A-B = A + ( - B )，因此我们只需要将B的只在输入加法器前变为-B即可，我们可以考虑补码：

    • B取反后加一

  • 在取反这个步骤中，我们可以想到我们的XOR逻辑门来实现，然后将C0设为1，即可实现减法器！

  • 可以分层实施muxes

  • ALU可以使用mux实现

    • 与基本块体单元耦合

  • N位加减器是使用N个1位加法器在输入端使用异或门完成的

    • XOR作为条件反相器
