GO基本语法解释

  • package指的是这个文件代码属于这个main包
  • import引入包
    • fmt: fmt包是Go语言标准库中的一个包，提供了格式化输入和输出的函数，例如Println、Printf、Sprintf等。这些函数可以用来打印文本、格式化字符串、读取用户输入等。
  • 没有分号

变量类型及初始化

1. 基本变量类型

Go 语言提供了几种基本数据类型：

  • 布尔类型：bool，取值为 true 或 false。
  • 数值类型：
    • 整数：int（系统默认大小）、int8、int16、int32、int64 以及它们的无符号版本 uint、uint8（即 byte）、uint16、uint32、uint64。
    • 浮点数：float32 和 float64。
    • 复数：complex64 和 complex128。
  • 字符串类型：string，不可变字符串。
  • 派生类型：
    • 指针：直接指向内存地址。
    • 数组：定长，类型相同的元素集合。
    • 切片：动态数组，更灵活，长度可变。
    • 映射（Map）：键值对的无序集合。
    • 结构体（Struct）：包含多个字段的复合类型。

2. 变量声明方法

Go 提供多种变量声明方式：

  • 标准声明：显式指定类型。

  • 类型推导：省略类型，由编译器根据赋值自动推导。

  • 简短声明：在函数内部使用 := 声明和初始化变量。

  • 多变量声明：可以同时声明多个变量。

  • 匿名变量：用 _ 表示匿名变量，不会被编译器存储，用于忽略某些返回值。

在 Go 中，未初始化的变量会有零值，整数为 0，浮点数为 0.0，布尔值为 false，字符串为 ""（空字符串），指针、切片、映射等为 nil。

切片

切片类似Java的List，Cpp中的vector

切片的基本属性

切片有三个基本属性：

  • 长度（len）：切片中元素的数量，使用 len(slice) 获取。
  • 容量（cap）：切片从起始索引到底层数组末尾的元素数量，使用 cap(slice) 获取。
  • 零值：未初始化的切片默认为 nil，长度和容量为 0。

切片是引用类型

切片是引用类型，传递切片时相当于传递指向底层数组的指针。对切片的修改会影响到其他引用相同底层数组的切片。

遍历map

当 range 用于 map 时，返回的是每个键和值。map 是无序的，因此遍历的顺序可能会每次都不同。

遍历字符串

当 range 用于字符串时，返回的是每个字符的字节位置（索引）和对应的 Unicode 值（rune）。
