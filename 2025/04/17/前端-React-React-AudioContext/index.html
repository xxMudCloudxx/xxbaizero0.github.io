🧠 什么是 AudioContext？

AudioContext 是 Web Audio API 的核心接口，它提供了一个音频处理的上下文环境，允许你创建、处理和播放音频。

在 AudioContext 中，音频通过一个音频图（Audio Graph）进行处理。每个节点（AudioNode）都可以接收、处理或输出声音。
