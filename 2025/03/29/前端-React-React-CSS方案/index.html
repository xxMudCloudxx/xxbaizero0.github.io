📌 基本用法列表

  1. 实用类（Utility Classes）
  2. 布局（Flex/Grid）
  3. 间距（Margin/Padding）
  4. 尺寸（Width/Height）
  5. 文本样式（Typography）
  6. 背景（Background）
  7. 边框（Borders & Radius）
  8. 伪类（Hover、Focus 等）
  9. 状态变体（Dark Mode、Active 等）
  10. 动画（Transitions & Animations）
  11. 响应式设计（Responsive Design）
  12. 复用样式（@apply & 自定义主题）

2️⃣ 实用类（Utility Classes）

Tailwind 提供了大量的类来快速开发，比如：

  • text-center → 文本居中
  • text-lg → 字体大小 lg
  • font-bold → 加粗
  • text-blue-500 → 颜色 blue-500

3️⃣ 布局（Flex/Grid）

✅ 使用 Flexbox

  • flex → 启用 display: flex
  • items-center → 垂直居中 (align-items: center)
  • justify-center → 水平居中 (justify-content: center)
  • h-screen → 高度 100vh

✅ 使用 Grid

  • grid → 启用 display: grid
  • grid-cols-3 → 三列
  • gap-4 → 间距 1rem

4️⃣ 间距（Margin/Padding）

  • m-4 → 外边距 1rem
  • p-6 → 内边距 1.5rem

更精细的控制

  • mt-2 → 顶部 margin 0.5rem
  • mb-4 → 底部 margin 1rem
  • ml-6 → 左侧 margin 1.5rem
  • mr-8 → 右侧 margin 2rem

5️⃣ 尺寸（Width/Height）

  • w-32 → 宽度 8rem
  • h-16 → 高度 4rem

6️⃣ 文本样式（Typography）

  • text-xl → 字体大小 1.25rem
  • font-semibold → 半粗
  • italic → 斜体
  • underline → 下划线

7️⃣ 背景（Background）

  • bg-gradient-to-r → 线性渐变
  • from-blue-400 → 起点颜色 blue-400
  • to-purple-600 → 终点颜色 purple-600

8️⃣ 边框（Borders & Radius）

  • border → 默认 1px 边框
  • border-red-500 → 红色边框
  • rounded-lg → 圆角 0.5rem
  • p-4 → 内边距 1rem

9️⃣ 伪类（Hover、Focus 等）

  • hover:bg-blue-700 → 悬停时变深色
  • text-white → 白色文本
  • py-2 px-4 → 适当的内边距

🔟 状态变体（Dark Mode、Active）

  • dark:bg-gray-800 → 深色模式背景 gray-800
  • dark:text-white → 深色模式文本白色

1️⃣1️⃣ 动画（Transitions & Animations）

  • transition → 过渡动画
  • duration-300 → 300ms 过渡
  • ease-in-out → 缓动函数
  • transform hover:scale-110 → 悬停时放大 1.1 倍

1️⃣2️⃣ 响应式设计（Responsive Design）

  • sm:text-lg → 小屏幕时 lg 大小
  • md:text-xl → 中等屏幕时 xl 大小
  • lg:text-2xl → 大屏幕时 2xl 大小
  • xl:text-3xl → 超大屏幕时 3xl 大小
